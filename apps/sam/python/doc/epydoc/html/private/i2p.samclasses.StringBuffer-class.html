Class StringBuffer

Deque --+
        |
       StringBuffer

A FIFO for characters. Strings can be efficiently appended to the end, and read from the beginning.

Example: 
>>> B = StringBuffer('Hello W')
>>> B.append('orld!')
>>> B.read(5)
'Hello'

>>> B.read()
'World!'
